# IN18
> This platform built using React JS ,Node Js ,Express Js,MongoDB.

## Features


* User

  - Add a User
  - Update a User
  - Delete a User


### Install Dependencies (frontend & backend)

```
cd front-end
npm install
cd backend
npm install
```

### Run

```
# in one terminal
npm  start (For frontend)
# in other terminal 
node index.js (For backend)
